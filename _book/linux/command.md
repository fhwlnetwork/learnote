# 系统管理的基础知识 

## 系统命令提示组成

```text
【root@hostname ~】# ---------命令提示符 
作用:只有在命令提示符后面输入命令才有效果 
组成： 
	1)登录用户的信息 
	2）@分隔符 
	3）主机名信息 
	4）当前所在系统的目录路径信息
```

## 系统命令是有语法规范

```tex
命令 参数 文件/路径 
命令与参数之间有空格分隔
```

## 系统目录结构简介

```text
linux目录结构从根开始 
绝对路径:从根开始查处文件 
缺点：寻找数据速度慢 
优点：准确性高 
相对路径：从当前位置 查找文件 
优点：找数据速度更快 
缺点：准确性低
```
